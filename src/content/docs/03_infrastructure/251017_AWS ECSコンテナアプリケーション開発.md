# 1. コンテナ基礎

## 1-1. コンテナの基本概念・構造

---

コンテナとは？

→ アプリケーションと、その実行に必要な**依存関係（周辺ソフトウェア）**をまとめてパッケージ化した技術

---

コンテナの構成要素

- OS: Linux のディストリビューションのみ指定可能 ※Linux カーネルはホストマシンのものを使用
- ランタイム
