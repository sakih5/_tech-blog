このページはアドモニション（情報ボックス）のスタイルをテストするためのページです。

## 基本的な使い方

Starlightでは以下の記法でアドモニションを使用できます。

### 情報 (Info)

:::note
これは情報ボックスです。
重要な情報や補足説明を記載します。
:::

カスタムタイトル付き：

:::note[インフォメーション]
カスタムタイトルを付けた情報ボックスです。
infoは省略可能です。
:::

### 警告 (Warning)

:::caution
これは警告ボックスです。
注意が必要な情報を記載します。
:::

カスタムタイトル付き：

:::caution[警告]
○○に注意してください。
:::

### より強い警告 (Alert/Danger)

:::danger
これはより強い警告ボックスです。
危険な操作や重要な注意事項を記載します。
:::

カスタムタイトル付き：

:::danger[より強い警告]
○○しないでください。
:::

### ヒント (Tip)

:::tip
これはヒントボックスです。
便利な情報やコツを記載します。
:::

## 使用例

### データベース接続の例

:::note[接続文字列]
PostgreSQLの接続文字列は以下の形式です：
```
postgresql://username:password@localhost:5432/database
```
:::

:::caution[パスワード管理]
本番環境では、パスワードを環境変数で管理してください。
ソースコードに直接パスワードを書かないでください。
:::

:::danger[SQL インジェクション]
ユーザー入力を直接SQLクエリに埋め込まないでください。
必ずプレースホルダーを使用してください。
:::

:::tip[パフォーマンス向上]
インデックスを適切に設定することで、クエリ速度が大幅に向上します。
:::
