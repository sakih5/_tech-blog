# React とは？

# `.tsx`ファイルとは？

TypeScript で React コンポーネントを書くためのファイルのことを`tsx`ファイルという。
React では、UI を定義するのに**JSX（JavaScript XML）**という文法を使用する。

```jsx
// JSX の例（.jsxファイル）
function Hello() {
  return <h1>Hello, world!</h1>;
}
```

JSX を使いつつ TypeScript で型安全に書きたい時に使うのが`.tsx`ファイルである。

```tsx
type Props = {
  name: string;
};

const Hello: React.FC<Props> = ({ name }) => {
  return <h1>Hello, {name}!</h1>;
};

export default Hello;
```

- `<h1>...</h1>`のように JSX 構文 を使っている。
- `Props`に型定義 (`name: string`) がある。

→ こういうファイルは .tsx でないと書けない。

## まとめ

React × TypeScript のプロジェクトでは、

- コンポーネントを書くファイル → .tsx
- ロジック・ユーティリティなど JSX を使わないファイル → .ts

| 拡張子 | 使う言語         | JSX（HTML っぽい構文）を使える？ | 主な用途         |
| ------ | ---------------- | -------------------------------- | ---------------- |
| `.js`  | JavaScript       | ❌ 使えない                      | 通常の JS コード |
| `.jsx` | JavaScript + JSX | ✅ 使える                        | React (JS 版)    |
| `.ts`  | TypeScript       | ❌ 使えない                      | 通常の TS コード |
| `.tsx` | TypeScript + JSX | ✅ 使える                        | React (TS 版)    |
