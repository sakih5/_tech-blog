## このサイトについて

**技術記事アーカイブ**は、個人開発やWeb技術の学習過程で得た知識を体系的にまとめた技術ブログです。

### 目的

- 学んだことを整理し、誰かの役に立つ形で共有する
- Web開発初心者〜中級者の方の学習をサポートする
- 自分自身の知識の定着と振り返りのため

### 対象読者

- Web開発を学び始めた初心者の方
- 特定の技術について詳しく知りたい中級者の方
- 個人開発に取り組んでいる方

### 扱っている技術

- **フロントエンド**: HTML, CSS, JavaScript, React, Next.js
- **バックエンド**: Python, FastAPI, API設計
- **インフラ**: WSL, Docker, AWS, CI/CD
- **データベース**: PostgreSQL, テーブル設計
- **ツール**: Git, VS Code, Linter/Formatter

### サイトの特徴

- **実践的**: 実際の開発経験から得た知識を共有
- **初心者に優しい**: 専門用語をできるだけ噛み砕いて説明
- **体系的**: カテゴリごとに整理された記事構成
- **継続的更新**: 新しい学びがあれば随時追加

### 技術スタック

このサイトは以下の技術で構築されています：

- **フレームワーク**: [Astro](https://astro.build/)
- **テーマ**: [Starlight](https://starlight.astro.build/)
- **ホスティング**: [GitHub Pages](https://pages.github.com/)
- **CI/CD**: GitHub Actions

### お問い合わせ

記事に関するご質問、誤字・脱字の報告、改善提案などは、[GitHubリポジトリ](https://github.com/sakih/_articles)のIssuesからお願いします。

### ライセンス

記事の内容は個人の学習と共有を目的として作成されています。
