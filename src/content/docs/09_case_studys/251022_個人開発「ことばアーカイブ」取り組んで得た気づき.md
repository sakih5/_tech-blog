# 個人開発「ことばアーカイブ」取り組んで得た気づき

## 本個人開発の目的

- AIコーディングエージェント（Claude Code）との協業の在り方を模索する
- Markdownファイルで要件定義書と設計書のドキュメントを作成する（脱エクセル）

## 使用技術スタック

フロント：Next.js + TypeScript
バックエンド API：FastAPI (Python)
DB：PostgreSQL
インフラ：AWS Lambda

## Step1 画面設計

以下を基本構成とすればOK（現時点で個人開発をしての感想）

- ログイン前ホーム
  - LPとして機能する
- ログイン画面
  - お知らせ（ここのバグを直した等、アップデートを記載。ログイン画面の左側にあると良い）
  - 利用規約・プライバシーポリシー（画面遷移で読めると良い）
- ホーム画面（ログイン後トップ）**★まずはここから考える**
  - ホーム画面で行う処理が大きい場合は、タブに分けたり、モーダルが出てくると良い
- 設定画面
  - アカウント設定（ユーザーネーム変更、アカウント削除等）
  - 問い合わせフォーム
